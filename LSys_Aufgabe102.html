Lineare Gleichungssysteme Aufgabe 102

Addiert man zwei Zahlen, so bekommt man 238. 
Die Zahlen verhalten sich wie 3 zu 11. 
Wie heißen die beiden Zahlen? 
 
x Erste Zahl 
y Zweite Zahl  
 
x + y = 238          (1) 
 
 x      3    
 -- = ----- 
 y     11 
 
Über Kreuz multipliziert 
 
11 * x = 3 * y | -3y 
 
11x - 3y = 0          (2)       
 
(1) * 3 + (2) 
 
 3x + 3y = 714 
11x - 3y =   0 
14x      = 714 |:14 
 
       x = 51
 
x = 51 in (1) eingesetzt 
 
51 + y = 238 |-51 
 
     y = 187
